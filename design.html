1. Color Palette & Design Tokens

The DMS ecosystem uses a cohesive, high-contrast palette. Bright functional colors highlight key statuses, stock numbers, and transaction values, while cool page backdrops ensure readability on mobile screen sizes.

2. Typography & Hierarchical Scale

The interface standardizes on the Poppins typeface for geometric precision and clarity across mobile screens. System symbols and localization strings use Inter or SF Pro respectively.

Role Font Family Weight & Style Size (Pixels) Context & Examples
Large Numeric Metrics Poppins Bold 20px - 24px Financial outstanding balances, large statistics (e.g. 97.5% Accuracy)
Section Headers Poppins SemiBold 16px - 18px Screen title headers (e.g. Stock Take, More Actions)
Card Titles & Products Poppins Bold / SemiBold 14px - 15px Tyre product brand names, selection action cards (e.g. By Brands)
Card Values & Inputs Poppins Medium / Regular 11px - 13px Quantity input values, stepper controls, form field helper labels
Timestamps & Alerts Poppins Medium 8px - 10px Audit intervals (e.g. Last audited: Yesterday), category items count

3. Layout & Spatial Radii

Stocktick and DMS modules operate on a responsive Mobile Grid designed for screen widths between 401px and 412px, featuring a standard outer margin of 16px. Desktop dashboard components are built for 1440px wide screens.

Standard Corner Radii Guide

  • Small Details (Status Badges, Stepper Buttons): 4px - 8px
  • Standard Cards & Action Buttons: 12px - 16px
  • Bottom Sheet Drawer Panels: 24px - 32px (Mixed top corners)
  • Illustrative Avatar Circles: 40px+

4. Page Architectural Breakdown

The Figma document describes an integrated ecosystem for tyre store managers and relationship managers, mapped across 10 main pages:

1. DMS Home Page ("DMS HOME PAGE") Canvas ID: 6256:44

The primary landing page welcoming users as "Hello, Tyre Czar". Supports regional translations (Kannada, Tamil, Telugu, Malayalam). Launches core operations: Buy Tyres, Manage Store, Pay, and Sell Tyres.

2. Inventory Dashboard ("inventory design final") Canvas ID: 2534:91

Comprehensive lists of in-hand stock filterable by vehicle category (Car, Bike, CV) and conditions (New, Used, Retread). Contains multi-step forms to add invoice items.

3. Stock Take Flow ("Stocktick") Canvas ID: 11358:1246

Physical audit module grouped by vehicle category or tyre brand. Features expected quantity comparisons, inline count confirm steppers, discrepancy logs, and a success confirmation overlay.

4. Ask Quote Bidding ("Ask_Quote_Dms") Canvas ID: 4906:2

B2B buying interface showing active live order bid collections, item variant sheets, cash splitting panels, and countdown timers.

5. Supplier Pricing ("Supplier_tp_dms") Canvas ID: 4025:36

Suppliers' bidding interface showing incoming quote notifications, pricing forms, rejection reason inputs, and bid approvals.

6. Reward Tiers ("Reward_screen") Canvas ID: 3775:537

Gamification layout detailing loyalty milestone badges (Silver, Gold, Platinum benefits) and reward earnings logs.

7. Order History & Outstandings ("DMS ORDER HISTORY") Canvas ID: 6453:348

Outstanding dues panel showing full/custom payment inputs, scheduled payment widgets, and bank confirmation cards.

8. Relationship Manager App ("RM APP") Canvas ID: 1190:65

App version for relationship managers. Features credit note claims (CN records), warranty raises, and dealer directories.

9. Next.js Desktop Dashboard ("Next js dashboard") Canvas ID: 3175:1229

Desktop web portal displaying statistics cards (Requests - 25, Quote Sent - 8623), list tables, and dual column layout grids.

6. Premium Design Recommendations

Implementing these visual suggestions will ensure the DMS application looks state-of-the-art:

  • Glassmorphic Drawers: Enable backdrop-filter: blur(12px) with a semi-transparent white background (rgba(255,255,255,0.85)) on bottom sheet headers.
  • Tactile Presses: Add scale-down spring motions on steppers (scale: 0.95) when users click the count increment buttons.
  • Blinking Live Bids: Apply subtle pulse keyframe animations on the active bid countdown badge in the Ask Quote page.
  • Tier Elevation Celebrations: Use lightweight particle effects (confetti canvases) when store managers unlock new Gold/Platinum tier benefits.
